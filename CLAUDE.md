# Criador de Site — Claude Code Config

## Projeto
Este repositório é usado para criação e desenvolvimento de sites.

## Stack preferida
- HTML, CSS, JavaScript (vanilla ou com frameworks leves)
- Node.js para scripts/build quando necessário

## Diretrizes gerais
- Escrever código limpo e bem estruturado
- Preferir soluções simples e diretas
- Comentar apenas onde a lógica não for óbvia
- Não adicionar dependências desnecessárias

## Skills disponíveis
- `nano-banana` — geração e edição de imagens
- `claude-api` — integração com a API da Anthropic/Claude
- `update-config` — configurar permissões e hooks do Claude Code
- `simplify` — revisar código gerado para qualidade e eficiência
- `find-skills` — descobrir e instalar novas skills

## Estrutura do projeto
```
Criador de Site/
├── CLAUDE.md          # Este arquivo — instruções para o Claude Code
├── src/               # Código-fonte dos sites
├── assets/            # Imagens, fontes, ícones
└── dist/              # Build final (gerado)
```
